# Matriz de afirmaciones públicas

Versión: 1 — 17 de julio de 2026

| Afirmación permitida | Evidencia requerida | Matiz obligatorio / afirmación prohibida |
|---|---|---|
| "La transcripción se ejecuta localmente" | Ruta de inferencia y prueba de red sin contenido | No decir que toda la aplicación es completamente offline: descarga modelos y comprueba actualizaciones |
| "El audio y las transcripciones no se envían por VoiceFlow" | Captura de red con frases canarias; revisión de código | Aclarar que copiar, pegar, exportar o publicar envía contenido al destino elegido por el usuario |
| "Sin cuentas, telemetría, analítica ni anuncios" | Inventario de dependencias y endpoints; prueba automatizada | Revalidar con cada dependencia nueva |
| "Usa modelos Whisper de IA" | Inventario de modelos, aviso inicial y licencias | No insinuar que la IA es infalible o humana |
| "Funciona offline después de descargar el modelo" | Prueba de arranque/transcripción con red deshabilitada | Actualizaciones, soporte y nuevos modelos requieren red |
| "Beta gratuita" | Flujo sin cobro y Términos vigentes | Bloquear antes de monetizar hasta completar la puerta comercial |

No publicar porcentajes de precisión, "100% exacto", "100% privado", "cumple GDPR/AI Act" ni garantías absolutas sin un protocolo documentado, evidencia reproducible y aprobación legal para el texto concreto.
